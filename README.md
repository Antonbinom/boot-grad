# boot-grad

Одностраничный сайт фирмы изготавливающей камины на заказ

## Функционал

- Адаптивная, валидная и кросбраузерная верстка
- Слайдеры
- Модальное окно
- Мобильное меню
- Плавный скролл
- Галерея
- Промо-ролик
- Фалидация формы
- Отправка формы на email

## Стек технологий

- HTML
- CSS
- SASS
- JS
- JQuery
- Плагины JQuery
- PHP Mailer
